# enhancing-ai-climate-change-ml
Using Machine Learning to enhance climate change resilience by predicting risks and suggesting mitigation strategies.
This repository contains the complete implementation of the project 
"Enhancing AI Climate Change Resilience Using Machine Learning". 

Included in this repo:
- **Code:** Python scripts and notebooks for data preprocessing, model training, and prediction.
- **Docs:** Flowcharts, architecture diagrams, and result graphs to explain the methodology.
- **Screenshots:** Images of project results and visualizations.

The project demonstrates how AI and ML can analyze climate data, predict risks, and suggest mitigation strategies, highlighting practical applications of machine learning for environmental resilience.
